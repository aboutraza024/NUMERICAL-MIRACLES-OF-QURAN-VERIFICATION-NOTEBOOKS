# NUMERICAL-MIRACLES-OF-QURAN-VERIFICATION-NOTEBOOKS

This repository provides **Google Colab notebooks** to reproduce and verify the numerical patterns discussed in the articles from:

https://numericalmiraclesofquran.com/

## 🎯 Purpose

The goal of this project is to:

- Convert article-based claims into **reproducible code**
- Allow users to **independently verify** numerical patterns
- Promote **transparency and scientific validation**
- Provide an open-source platform for further research
